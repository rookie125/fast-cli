## ifast-cli

> Quickly create react, Vue or applications without front-end framework

## Install
```bash
# yarn
$ yarn global add ifast-cli

# npm
$ npm install -g ifast-cli
```

## Usege

```bash
# Initialize new project
$ ifast create
```
